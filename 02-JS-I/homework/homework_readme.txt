1. En un archivo de texto separado que debes crear, escribe explicaciones de los siguientes conceptos como 
si se lo estuvieras explicando a un niño de 12 años. Hacer esto te ayudará a descubrir rápidamente cualquier 
agujero en tu comprensión.


 * Variables: Una Variable es un espacio donde se puede guardar informacion al igual que una caja donde guardas juguetes.

 * Strings: Un string lo podemos ver como las oraciones que escribimos en una hoja de papel.

* Funciones (argumentos, `return`): Una funcion la podemos describir como un procedimiento que el programa realiza 
 independientemente del resto del programa. Un argumento son valores que se le dan a estos procedimientos para que funcionen 
 de lo contrario no servirian de mucho. El return sirve para devolvernos los datos que le dimos una vez pasada por este 
 procedimiento.

 * Declaraciones `if`: Las declaraciones son la forma de como nosotros identificamos a las variables, constantes, etc de la 
 misma forma que le pones nombres a los juguetes para saber cual es cual. El if es un condicional, ¿Esto que quiere decir?
 quiere decir que son instrucciones que se pueden complir o no dependiendo de una condicion, un ejemplo claro seria 
 cuando vas al baño para laverte las manos, primero ves si tus manos estan sucias y en base a esa condicon decides si tienes 
 que lavarte las manos o no, el 'if' significa "si" se cumple esa condicion entonces se realiza la accion, en el caso anterior
 seria "si" tus manos estan sucias entonces te lavas las manos.

 * Valores booleanos (`true`, `false`): Es un tipo de dato que solamente tiene dos posibles opciones "true" o "false" 
 Se usa en situaciones especificas donde necesites ese tipo de respuesta o ese tipo de dato. Por ejemplo cuando tu madre 
 te pregunta si te lavaste las manos, tu le tienes que contestar con "si" o "no", nada mas que en este caso lo reemplazamos
 por true o falso (verdadero o falso).
